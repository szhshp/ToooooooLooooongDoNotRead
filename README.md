# ToooooooLooooongDoNotRead

## 开发背景

太长了不愿写

## 功能

使用快捷键阅读剪贴板里面的文字


1. 注册一个[  讯飞开放平台     ](https://www.xfyun.cn/)的账号
1. 在平台里创建一个 APP
1. 将其 API KEY 复制到软件的设置页面


## Debug

```
yarn
yarn run dev
```

## Build

可能需要管理员权限

```
yarn build
```
