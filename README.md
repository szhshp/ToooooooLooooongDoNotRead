# ToooooooLooooongDoNotRead

## 开发背景

太长了不愿写

## 功能

使用快捷键阅读剪贴板里面的文字


1. 注册 [讯飞开放平台账号](https://www.xfyun.cn/)
1. 创建一个 APP
1. 复制 API KEY


## Debug

```
yarn
yarn run dev
```

## Build

Prod 环境会关闭 Developer Tools 窗体

需要管理员权限

```
# build 全部环境
yarn build

# build 特定环境
yarn build:win32
```
